# 头像图片目录 / Avatar Images Directory

这个目录用于存放博客的头像图片和其他图片资源。

This directory is for storing avatar images and other image resources for the blog.

## 使用方法 / Usage

1. 将你的头像图片文件放到这个目录下
   
   Put your avatar image file in this directory

2. 在 `_config.yml` 中配置头像路径：
   
   Configure the avatar path in `_config.yml`:

   ```yaml
   avatar: /assets/img/your-avatar-filename.jpg
   ```

3. 建议图片尺寸：512x512 像素或更大，正方形格式
   
   Recommended size: 512x512 pixels or larger, square format

## 支持的图片格式 / Supported Image Formats

- JPG / JPEG
- PNG
- GIF
- WebP
